# Fuentes de entrada

Esta carpeta contiene archivos recibidos de fuentes institucionales antes de su
transformacion. Conservar nombres originales para mantener trazabilidad.

- `catalogos/`: docentes, tramites administrativos y lista oficial de
  resoluciones emitidas.
- `requisitos/`: anexo institucional de requisitos para maestro y doctor.
- `resoluciones/<anio>/`: ZIP o lotes originales de resoluciones firmadas.

Los resultados derivados no se guardan aqui. El lote 2026 genera CSV/JSONL en
`data/resoluciones_2026/`. No reemplazar una fuente existente sin registrar
fecha, procedencia y motivo.
